Ex 6.1

1.Điều gì xảy ra nếu người dùng chọn ‘Cancel’?
- Trong phương thức JOptionPane.showConfirmDialog, khi người dùng chọn cancel, giá trị trả về là JOptionPane.Cancel_OPTION. Trong code hiện tại, vì chỉ kiểm tra YES_OPTION và ngược lại in "No!", nên nếu chọn Cancel thì chương trình cũng in "No!".

2.Làm thế nào để tùy chỉnh các lựa chọn cho người dùng, ví dụ chỉ có hai lựa chọn: ‘Yes’ và ‘No’, hoặc ‘I do’ và ‘I don’t’?
- Có thể dùng JOptionPane.showOptionDialog để tự định nghĩa các nút lựa chọn. Ví dụ tạo mảng Object[] options = {"I do", "I don't"}; rồi truyền vào showOptionDialog. Khi đó hộp thoại sẽ hiển thị đúng các lựa chọn mong muốn thay vì mặc định Yes/No/Cancel.
Ví dụ:

import javax.swing.JOptionPane;

public class Ex1 {
    public static void main(String[] args) {
        Object[] options = {"I do", "I don't"};

        int choice = JOptionPane.showOptionDialog(
            null,
            "Do you agree?",
            "Custom Dialog",
            JOptionPane.YES_NO_OPTION,
            JOptionPane.QUESTION_MESSAGE,
            null,
            options,
            options[0]
        );

        if(choice == 0) {
            System.out.println("I do");
        } else if(choice == 1) {
            System.out.println("I don't");
        } else {
            System.out.println("Close dialog");
        }
    }
}
